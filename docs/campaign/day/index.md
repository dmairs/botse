Each game session is played over a number of rounds called days. Each day is divided into the five phases:

1. [New Day Phase](/docs/campaign/day/new-day-phase)
2. [Overland Phase](/docs/campaign/day/overland-phase)
3. [Encounter Phase](/docs/campaign/day/encounter-phase)
4. [Reward Phase](/docs/campaign/day/reward-phase)
5. [End of Day Phase](/docs/campaign/day/end-of-day-phase)
